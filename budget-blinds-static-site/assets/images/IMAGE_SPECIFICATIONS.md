## Image Optimization Tips

1. **Compress images** using tools like:
   - TinyPNG (tinypng.com)
   - Squoosh (squoosh.app)
   - ImageOptim (Mac)

2. **Use WebP format** when possible for better compression
   - Provide JPG fallback for older browsers

3. **Responsive images** consideration:
   - Consider creating 2x versions for retina displays
   - Hero: provide 1200px, 1600px, 2000px versions

4. **Lazy loading**:
   - Add `loading="lazy"` attribute to images below the fold
   - Hero image should load immediately (no lazy loading)
